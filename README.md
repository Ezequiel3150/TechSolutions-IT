# Servicio Técnico - Proyecto Pre-Entrega

Este es un proyecto web desarrollado para la pre-entrega del curso de Talento Tech. El propósito de esta página es ofrecer soluciones de Servicio Técnico, demostrando la aplicación de HTML semántico y diseño responsivo. 

Incluye una sección de servicios (productos) organizada con Flexbox, una grilla de reseñas de clientes utilizando CSS Grid, y un formulario de contacto funcional conectado a Formspree para la recepción de consultas.
